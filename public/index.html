bootstrab

bootstrab is a single-header anti-build system written in C++ for building C++. It provides serveral utilities for compiling programs efficiently along with the ability to rebuild itself on change. This project is also in development. I continue to add features and rework the library as I use it.

  • C++

ppgpu

ppgpu is a single header library for modern C++ that exposes bindings and provides modern abstractions with sane defaults over the wgpu-native library. With just this one header, users can start developing native GPU powered applications for desktop.
